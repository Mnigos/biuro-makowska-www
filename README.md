## Usage

### Setup

Make sure to have installed right version of Node.js in nvm

```bash
nvm use
```

Don't forget to install the dependencies:

```bash
yarn install
```

### Running the app

```bash
# Build app
yarn build

# Run app
yarn start

# Run app in watch mode
yarn dev
```

### Test

```bash
# unit tests
yarn run test

# unit tests in watch mode
yarn run test:watch

# Runing tests in coverage mode
yarn test:coverage
```
